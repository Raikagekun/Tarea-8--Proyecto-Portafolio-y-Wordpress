Galería de Imágenes

#UI #Diseño Web #HTML #CSS #Portafolio

Descripcion del proyecto

Galería de Imágenes fue un proyecto en el que tuvimos que codificar una página web con 12 imágenes, siguiendo un diseño establecido. La interfaz debía permitir, con un solo clic, acceder a enlaces como GitHub, Behance o correo electrónico. Además, la página debía ser completamente responsiva, adaptándose tanto a pantallas web como a celulares. Para desarrollarla, utilizamos HTML y CSS, trabajando en Visual Studio Code como editor de código.

Cliente

Tiempo de desarrollo

1 semana
